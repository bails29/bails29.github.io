Scripting for Environmental Scientists course!

Having a personal website seems to be the norm in academia, and has been something I have wanted to play around with for a little while now. I wanted to start by getting my hands dirty and writing my own HTML code rather than using a site generator, like Jekyll. for University of Manchester Scripting for Environmental Scientists course I edited several pages of this website, organized the file structure in my github repo, and created this new page! When I am done with it, I hope the projects blog will serve as a place to post about and showcase my ongoing projects. I originally used the github desktop app to work on this page, and thanks to the course am now keenly using bash! All pages were editted (at least in part) in the terminal.
